# Zotonye AI Video Generator

A simple web interface for generating AI comedy videos by Zotonye.

## Features
- Enter comedy script
- Click “Generate Video”
- Placeholder for future AI video generation backend

## How to Deploy on Render
1. Create a new GitHub repo.
2. Upload index.html, styles.css, script.js, README.md
3. Go to Render → Static Site
4. Connect repo
5. Build command: (leave empty)
6. Publish directory: .
